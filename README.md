# To-Do App

A simple, elegant, and fully responsive **To-Do List App** built using **HTML, CSS, and JavaScript**.  
It helps you manage tasks efficiently with progress tracking and local storage — so your tasks stay even after you refresh!

## Features

**Add, Edit, Delete Tasks** – Easily manage your daily to-dos  
**Progress Tracker** – Automatically updates as you complete tasks  
**Local Storage Support** – Tasks are saved in your browser  
**Responsive UI** – Clean, minimal, and mobile-friendly design  
**Keyboard Shortcut** – Press **Enter** to quickly add a new task

## Tech Stack

**HTML5** – Structure of the app  
**CSS3** – Modern, responsive, and minimal design  
**JavaScript (ES6)** – Dynamic functionality and localStorage integration
